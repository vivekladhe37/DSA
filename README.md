# 🧠 Data Structures & Algorithms in Python  
A structured, pattern‑based collection of LeetCode and FAANG‑style DSA problems solved in Python.  
This repository is organized by **patterns**, not random problem order — making it easy to revise, navigate, and scale as the number of problems grows.

## 🧩 Patterns Covered

### ✔ Arrays  
Basic array manipulation, sorting, frequency counting, and linear scans.

### ✔ Hashmaps  
Key‑value lookups, frequency maps, grouping, and anagram problems.

### ✔ Hashsets  
Duplicate detection, membership checks, and O(1) lookups.

### ✔ Two Pointers  
Sorted arrays, left/right pointer movement, pair/triplet problems.

### ✔ Prefix Sum  
Running totals, subarray sums, and hashmap‑based prefix logic.

### ✔ Sliding Window  
Dynamic window expansion/shrinking, substring problems, and frequency tracking.

### ✔ Stacks  
Monotonic stacks, parentheses problems, and next greater element.

### ✔ Binary Search  
Search in sorted arrays, boundaries, and optimization problems.

### ✔ Trees  
DFS, BFS, recursion, and binary tree traversal.

### ✔ Graphs  
Adjacency lists, BFS/DFS, cycle detection, and topological sort.

### ✔ Dynamic Programming  
Memoization, tabulation, subsequences, and optimization problems.

## 🛠 How to Use This Repo

1. Navigate to the pattern you want to study  
2. Open the Python file for the problem  
3. Review:
   - Approach  
   - Time & space complexity  
   - Edge cases  
   - Code implementation  
4. Practice similar problems in the same folder  
5. Use this repo as a revision tool before interviews  

---

## 🚀 Why Pattern‑Based Learning?

FAANG interviews don’t test random problems —  
they test **patterns**.

This repo helps you master:

- Sliding Window  
- Two Pointers  
- Prefix Sum  
- Hashmaps  
- Binary Search  
- Trees/Graphs  
- Dynamic Programming  

Once you understand the pattern, you can solve dozens of problems with the same technique.
